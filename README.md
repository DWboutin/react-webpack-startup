#Installation
##Local
```sh
$ npm install -g gulp-cli gulp babel babel-core babel-register
```

### Dev
After run this for the development env
```sh
$ npm start
```

Open another tab and run
```sh
$ npm run dev
```

### Prod
```sh
$ npm run prod
```

##Docker
```sh
$ docker-compose up
```

#Env
Node version: 6.5
Url: http://localhost:3000
